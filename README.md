# my-jobapp
This website demonstrates how to add, delete, edit, upload a file, upload multiple files, and search for data related to job openings. You need to sign up to obtain a username and password. Then, you can use the account you signed up for to add, delete, edit, upload a file, upload multiple files, and find data about job openings on the website, and use it whenever you want. The password you registered on the website will be encrypted using the bcrypt module to protect you from other users who may attempt to access your account.

https://my-jobapp.onrender.com/
